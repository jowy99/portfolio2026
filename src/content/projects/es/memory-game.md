## Resumen

Juego de memoria es una aplicacion sencilla pensada para entretener mientras ejercita la memoria. Se basa en una mecanica intuitiva y rapida, enfocada en ofrecer una experiencia ligera y agradable.

Desarrolle el proyecto desde cero, ocupandome tanto del diseno como del desarrollo front-end.

## Objetivos

1. Crear un juego simple y accesible para cualquier usuario.
2. Implementar una logica de memoria clara y estable.
3. Mantener una interfaz minimalista, limpia y funcional.
4. Ofrecer una experiencia fluida en navegacion y uso.

## Retos

No aparecieron bloqueos tecnicos grandes por el alcance reducido del proyecto. El foco principal estuvo en mantener la logica ordenada y evitar complejidad innecesaria.

Tambien priorice la claridad del flujo de juego para que el usuario entienda la dinamica desde el primer intento.

## Decisiones tecnicas

La implementacion se resolvio con JavaScript para la logica interactiva, HTML para estructura semantica y TailwindCSS para una capa visual rapida y coherente.

Se priorizo una arquitectura simple para facilitar mantenimiento y evolucion sin sobreingenieria.

## Highlights

- Interfaz sencilla y funcional orientada a usabilidad.
- Mecanica intuitiva para perfiles de usuario variados.
- Juego ligero, ideal para sesiones cortas de entretenimiento.
- Codigo implementado de forma integra por una sola persona.

## Resultados / Impacto

El proyecto quedo terminado y cumple su objetivo de entretenimiento ligero con una experiencia estable. Permitio practicar de forma aplicada patrones de interaccion en JavaScript.

Tambien reforzo criterios de diseno minimalista enfocado en lo esencial: claridad, respuesta y facilidad de uso.

## Lo que aprendi / Que haria mejor

Este proyecto me ayudo a fortalecer habilidades de JavaScript orientadas a logica de juego. Tambien confirme el valor de disenos simples cuando el objetivo es rapidez y buena experiencia.

En una siguiente iteracion anadiria niveles de dificultad, marcador de tiempo y estadisticas basicas de partidas.
